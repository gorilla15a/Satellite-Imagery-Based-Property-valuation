# property-valuation-multimodal
# Satellite Imagery-Based Property Valuation

This project builds a multimodal regression model to predict housing prices using:
- Tabular real estate features
- Satellite imagery extracted using latitude and longitude

## Structure
- `data/` : raw and processed datasets
- `notebooks/` : EDA and model training
- `src/` : helper scripts
- `outputs/` : figures and metrics
